# スマホ連絡メール作成ツール

---

## ファイル構成

| ファイル | 説明 |
|---|---|
| `checkin.html` | **スマホ専用** 連絡メール作成PWA |
| `manifest.json` | PWAマニフェスト（ホーム画面追加用） |

---

## 機能一覧


### checkin.html（スマホ専用PWA）

- 開始連絡 / 終了連絡 をワンタップで切替
- 体調・メンタルを **5段階ボタン** で選択
- 症状・一言・スタッフ共有事項を入力
- 送信先・件名・本文が自動生成され、「メールアプリで開く」で即送信
- お名前は次回起動時も保持（localStorage）
- ホーム画面に追加するとアドレスバーなしのアプリ表示
